Function
A block of code designed to perform a particular task.01 It can be defined using the function keyword, followed by the name of a function and parentheses. Optionally, a list of input parameters can be included within the parentheses. The code block that needs to be executed when the function is called is written within curly braces. A function expression in JavaScript is stored as a value and can be called anywhere in your program.
